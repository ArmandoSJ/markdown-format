## BACKSLASH ESCAPES
\*this is a comment\*
> Markdown provides backslash escapes for the following caracters:

* \Backslash 
* ()Parentheses 
* #Hash Mark 
* *Asterisk 
* +Plus sign
* _Underscore
* -minus sign(hyphen)
* {}Curly Braces
* .Dot
* []Square Brackets
* !Exclamation Mark
